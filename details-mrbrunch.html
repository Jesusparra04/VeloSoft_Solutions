MrBrunch: El sistema Pos para tu negocio gastronómico

MrBrunch está diseñado para pequeños y medianos restaurantes que desean mejorar su rapidez, organización y control. El sistema permite administrar ventas, inventarios, productos y personal desde una interfaz sencilla y muy visual.

¿Qué incluye MrBrunch?

  • Gestión de menús: Crea, edita y organiza productos fácilmente.
  • Control de pedidos: Vista detallada de órdenes activas y entregadas.
  • Inventario automático: Descuentos automáticos por venta.
  • Roles de usuarios: Administrador, cajero y cocina.
  • Reportes y estadísticas: Ventas, productos más vendidos y rendimiento.

Flujo de trabajo del sistema

  1. Registro del pedido: Se seleccionan productos y cantidades.
  2. Envío a cocina: La orden aparece automáticamente en la pantalla de cocina.
  3. Preparación: Cocina marca como “Listo”.
  4. Entrega al cliente: La orden cambia a “Entregado”.
  5. Cierre y pago: Se genera la factura simple o detallada.

Tecnologías con las que está desarrollado

  • HTML5, CSS3, JavaScript
  • Bootstrap
  • Python, Django
  • MySQL

¿Por qué elegir MrBrunch?

Interfaz intuitiva: Fácil de usar incluso para nuevos empleados.

Automatización: Reduce errores y mejora tiempos.

Reportes reales: Toma decisiones inteligentes para tu negocio.

Tiempo de implementación

Entre 2 y 4 días según el negocio. Incluye capacitación.
